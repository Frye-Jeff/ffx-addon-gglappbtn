# Google App Launcher

Simple access to Google Applications with a tool bar button.
This fork adds some minimal updates as the original project seems abandoned.



Currently has links to popular Google applications in a 3x3 grid. There is some customization support to pick the 9 apps and the order.

Supported apps:

* `gmail`: Gmail
* `gcal`: Calendar
* `gsheets`: Sheets
* `gdrv`: Drive
* `gnews`: News
* `gdoc`: Docs
* `gphotos`: Photos
* `gmaps`: Maps
* `gmeet`: Meet
* `gplay`: Play
* `gytube`: YouTube
* `gmsgs`: Messages
* `gemini`: Gemini
* `gcont`: Contacts
* `gtrans`: Translate
* `gkeep`: Keep
* `gmusic`: Music
* `gvoice`: Voice
* `gslides`: Slides
* `gclass`: Classroom
* `ggroups`: Groups
* `google`: Search
